Student Management System

A simple web-based Student Management System built using HTML, CSS, and JavaScript.  
This project helps manage student records such as adding, editing, deleting, and searching students.


Features

- Add Student
- Edit Student
- Delete Student
- Search Student
- Auto Student ID
- Data saved using Local Storage

 Technologies Used

- HTML
- CSS
- JavaScript



 Purpose of the Project
This project was created during my learning phase to understand:

- DOM manipulation
- CRUD operations
- Local Storage
- Basic web application development
